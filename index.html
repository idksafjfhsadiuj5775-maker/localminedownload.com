Features

How to Install

  1. Download the installer for your platform
  2. macOS: Open the DMG and drag to Applications. Right-click → Open on first launch (unsigned).
  3. Windows: Run the installer and follow the prompts.
  4. Linux: Make the AppImage executable: chmod +x Minecraft-Lite*.AppImage and run it.
  5. The game starts with its own built-in server — no setup needed.
